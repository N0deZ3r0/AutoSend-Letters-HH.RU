# Security Policy

## Supported Versions
Security updates are provided only for the latest version of this script.

| Version | Supported |
| ------- | --------- |
| Latest version | ✅ Yes |
| Any older version | ❌ No |

## Reporting a Vulnerability

**Please do NOT report security vulnerabilities through public GitHub Issues.**

If you believe you have found a security issue, please contact me directly by creating a new **Issue** on this repository and immediately marking it with the `security` label.

I will acknowledge your report within **48 hours** and provide a detailed response within **5 days**.

### What to Expect
*   **If the vulnerability is accepted:** I will work on a fix and release an update. You will be credited for the discovery (if you wish).
*   **If the vulnerability is declined:** I will explain why it is not considered a security issue.

## ⚠️ Important Security Note for Users
This script runs inside the **"Scripty" browser extension**, which has broad permissions. You are trusting both this script and the Scripty extension. Always install extensions from official sources.
